           RESUMES FOLDER - INSTRUCTIONS
═══════════════════════════════════════════════════════════════

PUT YOUR RESUME PDF FILES IN THIS FOLDER

Example:
  resumes/John_Doe_Resume.pdf
  resumes/Jane_Smith_DevOps_Resume.pdf


IMPORTANT:
──────────────────────────────────────────────────────────────
1. File must be PDF format (.pdf)
2. Update the path in config/config.json:
   "resume_path": "resumes/Your_Resume.pdf"
3. Keep filename simple (no spaces if possible)
   Good: John_Doe_Resume.pdf
   Bad:  My Resume (Final Version) 2024.pdf


RESUME TIPS FOR AUTOMATION:
──────────────────────────────────────────────────────────────
✓ Use keywords from job descriptions
✓ Keep it to 1-2 pages
✓ Use standard section headers:
  - Experience
  - Education
  - Skills
  - Projects (if relevant)
✓ Include specific technologies/tools
✓ Quantify achievements (numbers, percentages)


MULTIPLE RESUMES:
──────────────────────────────────────────────────────────────
You can store multiple versions:
- resumes/DevOps_Resume.pdf
- resumes/Cloud_Engineer_Resume.pdf
- resumes/SRE_Resume.pdf

Then switch in config.json based on your target role.
